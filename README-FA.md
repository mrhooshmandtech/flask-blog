# مراحل نصب 📌

## دانلود سورس کد از گیت هاب 💾
````
git clone https://github.com/mrhooshmandtech/flask-blog.git
````
## رفتن به پوشه 📁

````
cd flask-blog
````

## ایجاد محیط مجازی 🗃

````
virtualenv venv 
````

## فعالسازی محیط مجازی ✅

###  در ویندوز :
````
.\venv\Scripts\activate
````

### در لینوکس :
````
source \venv\bin\activate
````

## نصب پیش نیازها 📥
````
pip install -r requirements.txt
````

## و حالا اجرای برنامه فلسک 🆗
````
flask run
````
